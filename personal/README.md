# Nocsis Personal

個人向けのアプリです
PWA として、個人のスマホやパソコンから、授業やイベントの内容を見られるようになっています

## 使用している技術

### 言語/フレームワーク

- Flutter
- Dart

### ライブラリ

- go_router
  - ルーティングに使用
- riverpod
  - 状態管理やデータのキャッシュに使用
- freezed/json_serializable
  - フェッチしたデータの型チェックに使用

### デプロイ

- Vercel

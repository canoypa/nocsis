# Nocsis

自クラス向けに開発している、授業やイベントを確認するためのシステムです

このシステムは以下の 3 つのプロジェクトで構成されています

- [Classroom](/classroom)
  - 常時教室でその日の予定などを表示するディスプレイアプリ
- [Personal](/personal)
  - 個人向けの指定した日の予定や年間のイベントを見ることができるアプリ
- [Functions](/functions)
  - データの受け渡しや API 呼び出し等を行うバックエンド関数
